Model Settings

Starter Model

The starter model estimates peak rise time using the formula: t = k · inoculation−α · Q10(Tref−T)/10. Tune k and α after a few bakes to match your own starter.

Base rise time at reference conditions. Increase if your starter peaks later than predicted; decrease if it peaks earlier.

Inoculation sensitivity. Higher = more impact from changing seed ratio. Typical range: 0.6–1.0.

Temperature sensitivity — how much faster fermentation gets per +10 °C. Shared by both models. Typical value: 2.0.

Reference temperature at which k was measured. Set to the room temp you normally bake at.

Bulk Model

Bulk time is estimated as: t = c · (starter%)−β · Q10(26−T)/10. Adjust c and β to match your dough's actual bulk behaviour.

Base bulk time at 20% starter and 26 °C. Increase if your dough consistently needs more time.

Starter % sensitivity for bulk. Higher = bigger effect when changing inoculation. Typical range: 0.5–0.8.
